# MIO: Memories in Orbit datamining scripts
## extracted_strings
Contains extracted, localized text as raw, comma separated `txt`, converted into `json`.

## localization_extraction
Contains the scripts used to generate `extracted_strings`

## texture_extraction
Contains the scripts used to generate all `png` files found. Extracted results are in a [separate repo on the MIO-Modding org](https://github.com/MIO-Modding/extracted-textures).
